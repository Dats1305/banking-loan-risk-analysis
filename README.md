**Banking Loan Portfolio Performance and Risk Analysis**
Overview
This project analyzes a banking loan portfolio of 100,000+ records to assess credit risk exposure and portfolio performance.

Key Metrics:
Total Loans: 100K,
Total Loan Amount: 1,176.04 bn,
Default Rate: 22.64%,
Charged-Off Amount: 7.35 bn

Analysis Performed:
Loan status distribution,
Credit score risk segmentation,
Income impact analysis,
Bankruptcy risk exposure,
Interactive filtering by Term and Purpose

Tools Used:
Power BI,
Excel,
DAX
